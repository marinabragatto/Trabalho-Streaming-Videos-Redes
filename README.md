# Trabalho de Streaming de Vídeos Sob Demanda

### 1. Inicie o servidor
Em um terminal, abra a pasta server e execute o servidor:
```bash
cd server
go run server.go
```
### 2. Inicie o cliente
Em outro terminal, abra a pasta cliente e execute:
```bash
cd client
go run client.go
```
